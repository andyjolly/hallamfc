📋 Manual Instructions

Google Calendar

  1. Open Google Calendar
  2. Click the + next to "Other calendars"
  3. Select "From URL"
  4. Enter: https://andyjolly.github.io/hallamfc/2024-2025.ics
  5. Click "Add calendar"

Apple Calendar

  1. Open Calendar app
  2. Go to File → New Calendar Subscription
  3. Enter: https://andyjolly.github.io/hallamfc/2024-2025.ics
  4. Click "Subscribe"

Outlook

  1. Open Outlook Calendar
  2. Right-click on "Other Calendars"
  3. Select Add Calendar → From Internet
  4. Enter: https://andyjolly.github.io/hallamfc/2024-2025.ics

Other Calendar Apps

  1. Find the "Add Calendar" or "Subscribe to Calendar" option
  2. Enter the URL: https://andyjolly.github.io/hallamfc/2024-2025.ics
  3. Save or subscribe to the calendar
